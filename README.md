# ZTL — Zero-Trust Logic / логика нулевого доверия

Тернарная логика с принципом **«истина не выдаётся в кредит»**: третье
значение Z (zero-trust, «истина не заслужена») никогда не порождает T,
если T не вынуждена при любом классическом прочтении Z. Default deny,
перенесённый из безопасности в таблицы истинности.

Запуск (Python 3, без зависимостей):

```
python3 ztl.py        # таблицы + проверка опорных аксиом
python3 audit.py      # аудит тождеств: живые/павшие законы, MP, теорема жадности
python3 paradoxes.py  # лжец, карусель, мститель
```

Спецификация и все проектные решения — в `SPEC.md`. Родословная честная:
функционально ZTL — фрагмент логики Бочвара B3 (1938); своё — выбор
примитивов (7 клеток дельты) и порождающий принцип. Подробности в SPEC.

Проект возник 2026-07-10 из разбора парадокса лжеца; сосед по верфи —
VSPL (временная параконсистентная логика потоков, та же папка Epic World).
