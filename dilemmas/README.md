# dilemmas — classic hard questions through the core

The genre: take a question that tradition settles by **postulate** or a folk
certainty that grants itself a verdict, find its logical skeleton, mark the
unverified atom `Z`, and **run it — don't argue it**. Then read the
**warranty, not the verdict**:

- `hereditary` — earned; holds under every grounding;
- `until-verification` — held **on credit**; collapses at a grounding —
  and that collapse *is* the refutation;
- a **constant** (same output on every reading of the evidence) — a frame,
  not a fact: the test never reads the evidence.

Each file is a deterministic stand: the formalization is stated in words
first (the honest step — a bad skeleton refutes only a straw man), then the
cells are measured. The essay or aphorism, if one comes, is delivery; the
run underneath is the proof.

A dilemma that reaches a resolution moves to `solved/<name>/` — the stand
plus `RESOLUTION.md`, the record of what was actually settled (findings,
each on a run; the closing statement; the aphorism if one was earned).
